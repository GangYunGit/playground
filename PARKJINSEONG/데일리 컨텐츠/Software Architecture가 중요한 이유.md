# Software Architecture가 중요한 이유?

- 소비자의 니즈가 깐깐해짐 → msa가 아닌 monolithic이면 따라가기 힘듬
- 고급 개발자란??? → **설계 능력**도 갖춘 개발자
- 설계와 **운영 경험**도 중요함 → 설계한대로 운영이 되었는지를 경험하는 것이 정말 중요함
  → 주니어 때 신입 프로젝트에 많이 들어가 볼 것 → 시니어 때 경쟁력이 생김
